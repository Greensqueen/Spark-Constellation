# Spark Constellation

A tiny browser app that turns your skills, interests, and goals into an interactive constellation of project ideas.

It is designed as a playful GitHub portfolio project: no build step, no accounts, no backend, and no dependencies.

## Features

- Enter your background as skills, interests, and goals
- Generate project ideas from your own profile
- Explore an animated constellation canvas
- Save your profile locally in the browser
- Export a Markdown project brief for GitHub
- Works by opening `index.html` directly

## Quick Start

Open `index.html` in any modern browser.

## Project Structure

```text
spark-constellation/
```

## Good First Customizations

- Add more project templates in `script.js`
- Change the color palette in `styles.css`
- Add a shareable image export for the constellation
- Turn exported briefs into GitHub issue templates
